───────────────────────────────────────────────────────────────────────────────
TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

If script fails to run:

WINDOWS (run-migration.bat):
  Problem: "curl is not recognized"
  Solution: Install git for Windows (includes curl) or use WSL

  Problem: Git push fails
  Solution: Check GitHub credentials: git config --list
  Solution: Try: git push -u origin main

MAC/LINUX (run-migration.sh):
  Problem: "Permission denied"
  Solution: chmod +x run-migration.sh

  Problem: Node not found
  Solution: Install Node.js: brew install node (macOS) or apt install nodejs (Linux)

If deployment fails:

  Check Vercel:  https://vercel.com/tccollectibles/financial-planner/deployments
  Check CF:      https://dash.cloudflare.com/
  Check Supabase: https://supabase.com/dashboard/

  Smoke test endpoints manually:
  - Vercel:     curl https://financial101vercel.app/api/admin/users
  - Cloudflare: curl https://financeplan-th.pages.dev/api/admin/users
