# Investment Expert Advisor

**ID:** investment-expert

**Category:** Investment Strategy

## Description

Delivers portfolio optimization, asset allocation strategy, and investment risk analysis. This expert advisor helps you:

- Build optimal asset allocation based on risk profile
- Analyze portfolio concentration and diversification
- Project investment returns under various scenarios
- Balance risk and return for your time horizon
- Evaluate investment vehicle selection

## When to Use

Use this skill when you:
- Need asset allocation guidance
- Want to rebalance portfolio
- Seek investment return projections
- Need risk assessment of holdings
- Are building portfolio from scratch
- Want to optimize for tax efficiency

## Core Functions

### optimizeAssetAllocation(riskProfile, currentAssets)
Recommends asset allocation percentages:
- Conservative: 30% stocks / 70% bonds
- Moderate: 60% stocks / 40% bonds
- Aggressive: 80% stocks / 20% bonds
- Time-horizon adjustments (target-date funds)

### analyzePortfolioRisk(holdings)
Evaluates portfolio risk metrics:
- Concentration analysis (largest position %)
- Diversification index
- Volatility estimate
- Currency exposure
- Sector concentration
- Drawdown scenarios

### projectInvestmentReturns(strategy, timeframe)
Projects returns under multiple scenarios:
- Base case (historical average)
- Best case (90th percentile)
- Worst case (10th percentile)
- Inflation-adjusted returns
- Contribution impact
